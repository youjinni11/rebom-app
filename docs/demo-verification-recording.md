# GIF 2 신원검증 — 녹화 가이드

데모 페이지 주소: `http://localhost:3000/demo/verification`

녹화용(컨트롤 숨김): `http://localhost:3000/demo/verification?record=1`

## 흐름 (8화면)

| 순서 | 내용 | 표시 시간 |
|------|------|-----------|
| 1 | 약관 및 동의 → 본인인증 하기 (자동 탭) | 2초 |
| 2 | 전화번호 본인인증 | 1.6초 |
| 3 | 본인인증 완료 | 1.6초 |
| 4 | 얼굴 인증 | 1.6초 |
| 5 | 얼굴 인증 완료 | 1.6초 |
| 6 | 미혼 상태 인증 → 나중에 하기 (자동 탭) | 1.6초 |
| 7 | 나중에 인증하기 안내 | 2초 |
| 8 | 2/3 완료 + 서비스 둘러보기 | 3초 |

**총 길이: 약 15초**

## 1. 개발 서버 실행

```bash
npm run dev
```

## 2. 브라우저에서 데모 열기

1. `http://localhost:3000/demo/verification?record=1` 접속
2. **반복 재생** 체크 **끄기** (1회만 재생)

## 3. 화면 녹화 (Mac)

1. `Cmd + 0` → 확대 100%
2. `Cmd + Shift + 5` → 세로 크림 화면만 선택
3. 약 **16초** 녹화 (여유 있게)

## 4. MP4 → GIF 변환

1. [ezgif.com/video-to-gif](https://ezgif.com/video-to-gif)
2. 권장: 너비 480~720px, **15 fps**
3. Convert to GIF → 다운로드

## 5. PPT에 넣기

- GIF 1 다음 슬라이드에 배치
- `?record=1` 로 녹화해 크림 화면만 사용
