# 가치관라이프 추가 플로우 스펙

> 질문·선택지 문구는 아래를 그대로 사용하세요. (임의 변경 금지)  
> 스펙과 화면 미리보기가 다르면 **스펙 우선**.

## 진입

프로필 탭 → 가치관라이프 → 「추가」(또는 「수정하기」)

## 공통 규칙 — 모든 질문 화면에 동일 적용

1. **복수 선택** 가능
2. **「+ 직접 추가」** 제공
   - 목록에 없는 답을 직접 입력 → 확인 시 칩으로 추가되며 선택 상태
3. 안내 문구를 **작게** 표시:

   > 정확하고 솔직하게 답변할수록 회원님께 더욱 잘 맞는 상대를 추천해드릴 수 있어요

4. **소비관 화면은 만들지 않음** (제외)

## 권장 UI 구성

`STEP 라벨` → `화면 제목` → `안내 문구(작게)` → `질문` → `선택 칩(복수)` → `「+ 직접 추가」` → `「다음」`

- 최소 1개 이상 선택 시 「다음」 활성화 권장

## 화면 순서

```
프로필 → 가치관라이프 「추가」
→ 1 정치성향
→ 2 가족관계
→ 3 종교
→ 4a 건강습관 · 건강 챙기기
→ 4b 건강습관 · 술·담배
→ 5 취미
→ 완료
```

참고: 기존 앱은 가치관/라이프스타일이 분리되어 있음.  
이번 요청은 위 순서로 통합된 「가치관라이프」 플로우 기준.

---

## [1] 정치성향

**질문:** 「정치·사회 이슈에 대한 생각은 어떠신가요?」

**선택지** (복수 + 직접추가):

- 보수에 가까워요
- 진보에 가까워요
- 중도에 가까워요
- 잘 모르겠어요 / 관심 없어요
- 다르면 존중하며 대화해요

## [2] 가족관계

**질문:** 「지금 함께 지내는 가족·생활은 어떤가요?」

**선택지** (복수 + 직접추가):

- 혼자 지내고 있어요
- 자녀와 함께 살아요
- 자녀는 독립했어요
- 반려동물과 살아요
- 부모님과 함께 살아요
- 친구와 가까이 지내요

## [3] 종교

**질문:** 「신앙·종교에 대해 어떤 편이신가요?」

**선택지** (복수 + 직접추가):

- 기독교를 믿어요
- 천주교를 믿어요
- 불교를 믿어요
- 특정 종교는 없어요
- 종교 얘기는 잘 안 해요
- 다르면 존중하며 대화해요

## [4a] 건강습관 · 건강 챙기기

**질문:** 「평소 건강은 어떻게 챙기시나요?」

**선택지** (복수 + 직접추가):

- 꾸준히 운동해요
- 산책·걷기를 즐겨요
- 식단을 신경 쓰는 편이에요
- 규칙적인 수면을 중요하게 여겨요
- 건강검진을 챙기는 편이에요
- 특별한 습관은 없어요

## [4b] 건강습관 · 술·담배

**질문:** 「술·담배는 어떤 편이신가요?」

### 음주 (복수 + 직접추가)

- 나는 거의 안 마셔요
- 나는 맛있는 자리엔 한잔 정도
- 나는 즐겨 마시는 편이에요
- 상대는 안 마셨으면 해요
- 상대의 가벼운 음주는 괜찮아요
- 함께 한잔해도 좋아요

### 흡연 (복수 + 직접추가)

- 나는 피우지 않아요
- 나는 가끔만 피워요
- 나는 평소에 피워요
- 상대는 안 피웠으면 해요
- 상대가 피워도 괜찮아요
- 크게 상관없어요

## [5] 취미

**질문:** 「평소 즐기시는 취미는 무엇인가요?」

**선택지** (복수 + 직접추가):

- 외국어 학습
- 독서
- 요리
- 헬스, 요가, 필라테스
- 투자공부
- 러닝·수영
- 축구·야구·테니스·골프

선택한 내용은 프로필에 반영하고, 매칭 추천에 사용.
