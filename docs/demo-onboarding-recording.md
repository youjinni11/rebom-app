# GIF 1 온보딩 — 녹화 가이드

데모 페이지 주소: `http://localhost:3000/demo/onboarding`

녹화용(컨트롤 숨김): `http://localhost:3000/demo/onboarding?record=1`

## 1. 개발 서버 실행

프로젝트 폴더에서 아래 명령을 복사해 터미널에 붙여넣으세요.

```bash
npm run dev
```

## 2. 브라우저에서 데모 열기

1. Chrome 또는 Safari에서 `http://localhost:3000/demo/onboarding` 접속
2. **반복 재생** 체크를 **끄세요** (1회만 재생 → GIF 길이 약 18~20초, 자동 탭·클릭 포함)
3. 우측 상단 컨트롤은 녹화 화면에 안 보이게, 화면 밖으로 드래그하거나 녹화 후 잘라내세요

## 3. 화면 크기 (9:16 세로)

데모 화면은 **세로 휴대폰 비율(9:16)** 입니다 (720×1280과 같은 비율).

### 녹화할 때 (추천)

`?record=1` 주소를 쓰면 **크림색 세로 화면만** 보입니다. 어두운 녹색 여백이 없어서 녹화가 쉽습니다.

1. 브라우저 창을 **최대한 크게** 키우기
2. `Cmd + 0` 으로 확대 **100%** 맞추기
3. `Cmd + Shift + 5` → **세로로 긴** 크림 영역만 선택 녹화

### 미리볼 때

일반 주소(`?record=1` 없음)는 가운데 세로 박스 + 바깥 어두운 녹색 여백으로 보입니다.  
녹화본에 여백이 들어가지 않게 **`?record=1` 로 열어서 녹화**하세요.

## 4. 화면 녹화 (Mac)

### 방법 A — `?record=1` + 전체 화면 (가장 쉬움)

1. `http://localhost:3000/demo/onboarding?record=1` 접속
2. `Cmd + Shift + 5` → **세로로 긴** 크림색 화면만 드래그해서 선택
3. 약 **20초** 녹화

### 방법 B — 영역 선택

1. `Cmd + Shift + 5` → **선택 영역 녹화**
2. **세로가 더 긴** 크림색 직사각형만 드래그 (가로로 긴 화면이면 잘못 선택한 것)
3. 약 **20초** 녹화

## 5. MP4 → GIF 변환

1. [ezgif.com/video-to-gif](https://ezgif.com/video-to-gif) 접속
2. 녹화한 MP4 업로드
3. 권장 설정:
   - **Size**: 너비 480~720px (세로 GIF, PPT에 맞게 조절)
   - **Frame rate**: 15 fps
4. **Convert to GIF** → 다운로드

## 6. PPT에 넣기

- 슬라이드 한 칸에 GIF 하나
- 배경이 어두운 녹색(`#1a2522`)이면 PPT 배경색과 맞추거나, GIF 주변을 잘라 크림 화면만 보이게 하세요
